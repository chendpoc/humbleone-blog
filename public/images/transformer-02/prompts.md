## 4. cross-entropy-curve.png — Cross Entropy 曲线

**对应章节**：9.2 公式 Loss = -ln(p_correct)

**提示词**：

> 一张中文数学教学插图，展示交叉熵损失函数 y = -ln(x) 的曲线。横轴 x 标注"模型分配给正确答案的概率 p"，范围 0 到 1。纵轴 y 标注"Loss = -ln(p)"。
>
> 曲线从左上角陡峭下降到右下角：
>
> - 在 x ≈ 0 附近，y 趋近 +∞（红色渐变区域，标注"预测完全错误，惩罚极重"）
> - 在 x ≈ 0.5 附近，y ≈ 0.69（橙色区域，标注"一半把握，中等惩罚"）
> - 在 x ≈ 0.85 附近，y ≈ 0.16（黄绿区域，标注"比较确定，惩罚较小"）
> - 在 x → 1 处，y → 0（绿色区域，标注"完全正确，loss = 0"）
>
> 曲线上标注出这四个特征点，用小圆点标记。曲线颜色从红色（左）渐变到绿色（右）。浅灰网格，白色背景，学术风格。宽高比 4:3。

**想要传达的**：越接近正确答案奖励越陡、越远离惩罚越重——对数放大了这个效应。
